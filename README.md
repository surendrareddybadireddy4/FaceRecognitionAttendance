# Face Recognition Attendance System

## Overview

A Face Recognition Attendance System developed using Python, OpenCV, and LBPH Face Recognizer. The system automatically detects and recognizes faces through a webcam and records attendance while preventing duplicate entries.

## Features

* Face Detection using OpenCV
* Face Recognition using LBPH
* Automatic Attendance Marking
* Duplicate Attendance Prevention
* Attendance Storage in CSV

## Technologies Used

* Python
* OpenCV
* NumPy
* LBPH Face Recognizer

## Project Workflow

1. Capture face images
2. Train LBPH model
3. Recognize face using webcam
4. Mark attendance automatically
5. Store attendance in CSV
